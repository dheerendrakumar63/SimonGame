# SimonGame 
I create a SimonGame , I used HTML , CSS AND JAVASCRIPT  during it making.

SimonGame: A SimonGame is a memory-based game where:

1. The computer shows a sequence of colors.

2. The player must repeat the sequence in the correct order. 

3.Each level adds one more color to the sequence.

4. if the user presses a wrong color -> Game Over!



Featured Covered:

1. Random sequence generation

2.User input checking

3.Level progression 

4. Game Over screen.

5.Button animation
